# 🎮 Tic Tac Toe Game

A simple and interactive **2-Player Tic Tac Toe** game built using **HTML, CSS, and JavaScript** in a single file. Players take turns placing **X** and **O** on a 3×3 grid. The game automatically detects a winner or a draw and allows restarting the game at any time.

---

## 📌 Features

- 👥 Two-player mode
- 🎨 Modern and responsive user interface
- ✨ Glassmorphism design
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔄 Restart game button
- 🎉 Winning cells highlighted with animation
- 📱 Works on desktop and mobile browsers

---

## 🎮 How to Play

1. Player **X** starts the game.
2. Players take turns clicking an empty cell.
3. The first player to align **three identical symbols** horizontally, vertically, or diagonally wins.
4. If all nine cells are filled without a winner, the game ends in a draw.
5. Click the **Restart Game** button to start a new match.

---

## 🧠 Game Logic

- Uses a **3×3 array** to store the game board.
- Checks all possible winning combinations after every move.
- Prevents players from selecting already occupied cells.
- Stops the game immediately after a win or draw.
- Resets the board when the Restart button is clicked.
